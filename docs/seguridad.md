# Seguridad

## MFA
Autenticacion multifactor para cuentas administrativas y de alto riesgo.

## JWT
Control de acceso mediante tokens firmados y renovacion con refresh token.

## Roles
Modelo RBAC para segmentar permisos por perfil operativo.

## Auditoria
Bitacora de acciones criticas sobre usuarios, expedientes, documentos y facturacion.

## Encriptacion
- Datos en transito con HTTPS/TLS
- Cifrado de secretos y llaves
- Hash de credenciales

## Backups
Respaldos cifrados y politicas de retencion por cumplimiento.

## Gestion de sesiones
- Expiracion de sesiones
- Revocacion de refresh tokens
- Deteccion de actividad anomala

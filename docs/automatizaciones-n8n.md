# Automatizaciones n8n

## Flujos
- WhatsApp: recordatorios de audiencias, tareas y vencimientos
- Correo: notificaciones de cambios, seguimiento y estados
- Agenda: sincronizacion de eventos y alertas
- Cobros: avisos de facturas y seguimiento de mora
- Seguimientos: tareas automáticas por etapa de expediente
- Marketing: nutricion de prospectos y campañas segmentadas
- Expedientes: disparadores por actualizacion de estado

## Patrones tecnicos
- Webhooks entrantes desde API
- Jobs programados
- Integracion con plantillas y colas de envio
- Registro de ejecucion para auditoria y reintentos
